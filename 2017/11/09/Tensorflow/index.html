Tensorflow tutorials

tensor

Tensorflow的中心数据单元是tensor。Tensor是由一组以任意维数所组成的矩阵组成的，Tensor的等级就是维数。

3   # a rank 0 tensor;
[1., 2., 3.] # a rank 1 tensor; a vector with shape [3]
[[1.,2., 3.], [4., 5., 6.]] # a rank 2 tensor; a matrix with shape [2, 3]
[[[1., 2., 3.]], [[7., 8., 9.]]] # a rank3 tensor with shape [2, 1, 3]

TensorFlow Core tutorial

在机器学习中，我们通常需要一个可以采取任意输入的模型，比如上面的模型。 为了使模型可训练，我们需要能够修改图形以获得具有相同输入的新输出。Variables 可以使得我们向图添加可训练的参数。

W = tf.Variable([.3], dtype=tf.float32)

在我们使用tf.Variable时我们首先要对其进行初始化。

init = tf.global_variables_initializer()
sess = tf.Session()
sess.run(init)

我们可以使用tf.square计算每一个样本误差，通过_tf.reducesum函数来计算总误差。

y = tf.placeholder(tf.float32)
squared_deltas = tf.square(linear_model-y)
loss = tf.reduce_sum(squared_deltas)

我们可以通过tf.assign函数来改变由tf.Variable函数所赋值的变量。

fixX = tf.assign(W,[-1.])

tf.estimator 是一个高级的Tensorflow库，用来简化机器学习的机制。

  • running training loops
  • running evaluation loops
  • managing data sets

tf.feature_column.numeric_column 这个函数是用来创建有几个feature的，本例中只有一个feature。

tf.feature_column.numeric_column("x", shape = [1])

通过tf.estimator直接构建分类器，但是数据load进来之后，输入分类器之前我们先要使用input_fn的函数。如果你的feature/label数据是python array，或者存在pandas dataframe或者numpy array中，需要使用如下函数建立input_fn函数。

tf.estimator.inputs.pandas_input_fn()
tf.estimator.inputs.numpy_input_fn()

tf.train.get_global_step(graph = None)

  • Function：获得全局步张量。并且全局的步张量必须是整形变量。

tf.assign_add()

assign_add(
  ref,
  value,
  use_locking = None,
  name = None
  )

  • Function:更新ref通过将value值加给它。

  • Args:

    - ref: 一个可修改的Tensor
    • value: 一个和ref相同类型的Tensor
    • use_locking: bool型。
    • name: 给该操作起名称。

MNIST For ML

交叉熵函数:用来确定模型的损失。

cross_entropy = tf.reduce_mean(-tf.reduce_sum(y_ * tf.log(y),reduction_indices = [1]))
  • reduction_indices的作用是将tf.reduce_sum计算出的值存在新一列。

但是，在源码当中我们并不使用他，因为他的计算不够稳定，我们使用如下的函数：

tf.nn.softmax_cross_entropy_with_logits

tf.argmax:gives you the index of the highest entry in a tensor along some axis.

tf.argmax(y_, 1)

  • 表示正确的标签。

    tf.argmax(y, 1)

  • 表示我们训练的模型预测出的标签。

tf.equal:用来检查我们的标签和真实标签的,返回booleans.

tf.equal(tf.argmax(y,1), tf.argmax(y_, 1))

tf.cast:将tf.equal函数得到的，booleans值，转化成数值型。

tf.Session()和tf.InteractiveSession()区别

  • tf.Session()在执行变量时需要sess.run(x)
  • tf.InteractiveSession()执行时仅仅需要x.eval()即可。

Name Scopes

当我们在处理复杂的模型时，图很容易因为许多的结点而变得很混乱。为了避免这个，我们可以使用name space,可以把有关的结点放在一起。

with tf.name_scope('name')
